# React Weather App

This project is built using **React v18** and **Node.js 22.11.0**.

### To get started:

1. cd into react-weather-app in a new terminal if you havent already:

   ```bash
   cd react-weather-app

   ```

2. Install the correct version of React:

   ```bash
   npm install react@18 react-dom@18

   ```

3. make sure you get your api keys and put them in your .env

   https://openweathermap.org/

   https://rapidapi.com/wirefreethought/api/geodb-cities



4. Then continue with the regular setup:

   ```bash
   npm install
   ```

   ```bash
   npm start
